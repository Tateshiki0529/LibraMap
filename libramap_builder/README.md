# LibraMap Builder

蔵書DB生成・検証・デバッグ補助ツール（開発者・管理者向け）。

## 概要

LibraMap Builder は本番運用システムではなく、
開発・検証・保守支援を目的とした補助ツール群です。
通常の司書業務では使用しません。

## 主な用途

- ISBN 大量投入によるローカル蔵書 DB の初期構築
- NDL Search API の動作検証
- テストデータの生成
- キャッシュ・NDC の内容確認
- CSV インポート検証
- デバッグ支援

## 運用方針

Builder で生成したデータはレビュー・確認後に本番 DB へ反映します。
Builder 側から直接本番 DB を書き換えないことを推奨します。

仕様参照: specs.md §3.0.1〜3.0.4
