# LibraMap Editor

書架マップJSONを編集する管理者向けツールです。

## 起動

```powershell
python -m libramap_editor.main
```

標準では `libramap/data/schema.json` を開きます。

## MVP機能

- フロアの追加・削除
- 書架、壁、階段、カウンター、禁帯出、返却BOXなどの追加・削除
- オブジェクトID、種別、座標、サイズの編集
- 書架の段数・列数編集
- 書架ごとのNDC範囲セグメント編集
- JSON保存、別名保存、検証
- 編集中フロアのプレビュー表示

## 運用メモ

- 保存前に `検証` を実行してください。
- 実運用DBへ反映する前に、LibraMap本体で配架判定とレシート表示を確認してください。
- ドラッグ配置とリサイズは次フェーズで対応予定です。
