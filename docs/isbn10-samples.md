# ISBN-10 Sample Set

Issue #3 (ISBN-10入力対応) の手動確認用サンプルです。

## 基本サンプル
| ISBN-10 | 期待されるISBN-13 |
|---|---|
| 4101010110 | 9784101010113 |
| 123456789X | 9781234567897 |

## 追加サンプル（検証用）
| 種別 | 入力 | 期待 |
|---|---|---|
| ISBN-10 正常 | 4873113369 | ISBN-13に変換されて処理継続 |
| ISBN-10 正常(X終端) | 123456789X | ISBN-13に変換されて処理継続 |
| ISBN-10 不正 | 4101010111 | バーコード判定不可エラー |

## 補足
- 入力時のハイフンや空白は除去されます。
- `X` はISBN-10のチェックデジットとして末尾のみ有効です。
